# Ezgi Kafkas — Portfolio

Figma tasarım sistemine göre hazırlanmış, responsive ve GitHub Pages uyumlu tek sayfa portfolyo.

## Yayınlama

1. Bu klasörü yeni bir GitHub reposuna yükleyin.
2. GitHub'da **Settings → Pages** bölümünü açın.
3. **Deploy from a branch**, ardından `main` ve `/ (root)` seçin.
4. Site birkaç dakika içinde `https://KULLANICI-ADINIZ.github.io/REPO-ADI/` adresinde yayınlanır.

Yerelde açmak için `index.html` dosyasını tarayıcıda açmak yeterlidir.

## Bağlantıları kişiselleştirme

`index.html` içindeki e-posta, Behance, LinkedIn ve Instagram adreslerini kendi profil linklerinizle değiştirin.
